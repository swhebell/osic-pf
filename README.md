# osic-pf
OSIC Pulmonary Fibrosis Progression Kaggle

DICOMtests.ipynb - CT importation, lung masking, pre-processing and calculation lung tissue volume

TabularTest.ipynb - pre-processing of tabular data and trial run of MLP

3DCNNtest.ipynb - 3DCNN prototyping with CT data only

CombinedTest.ipynb - combination of CT and tabular data to predict FVC progress
